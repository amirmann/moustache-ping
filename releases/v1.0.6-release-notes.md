# Moustache Ping v1.0.6

## Fixes

### Network Info — DNS & Cellular
- DNS, gateway, subnet, and IPv6 are now read via native Android `ConnectivityManager.getLinkProperties()` instead of shell commands (which don't work inside the app sandbox)
- Active network is preferred; Private DNS hostname is shown when system DNS servers are empty
- Cellular card now shows IPv6, subnet, gateway, and DNS when available

### Subnet Scan — Diff between scans
- Scan state is kept when switching tabs (removed autoDispose)
- Second scan automatically compares against the previous scan — no need to save a baseline first
- Diff card shows Added/Removed hosts with counts and source label
- Hint shown after first scan: "Scan again to compare with this result"

## Downloads
- `moustache-ping-v1.0.6-arm64.apk` — most phones (recommended)
- `moustache-ping-v1.0.6-armeabi-v7a.apk` — older 32-bit devices
