# Moustache Ping v1.0.9

## Changes

- **Removed Copy MAC** — Android does not expose ARP MAC addresses to apps, so the option was always unavailable
- **Screenshots on GitHub** — light + dark theme screenshots (fictional `DemoNet` demo data only)
- **Screenshot generator** committed: `test/screenshot_capture_test.dart` + `./tool/capture_screenshots.sh`
- **arm64-only APK** — no longer builds or ships `armeabi-v7a`
- Narrow-layout fixes for Ping / Speed control rows
- **Scan diff readability** — darker green/red on light theme; IP under hostname uses normal secondary text color instead of washed-out accent

## Download

- `moustache-ping-v1.0.9-arm64.apk` — recommended for modern phones
