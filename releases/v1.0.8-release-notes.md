# Moustache Ping v1.0.8

## Changes

### Network Info — DNS display
- Strip IPv6 zone IDs (e.g. `%wlan0`) that made DNS look broken
- Show each DNS server on its own line
- Prefer IPv4 DNS first

### Settings — Light theme default
- New **Settings** tab
- Light theme is now the default
- Dark theme toggle (preference saved)

### Subnet Scan — Long-press copy
- Long-press a host in the list or in the Added/Removed diff to copy:
  - Hostname (when available)
  - IP address
  - MAC address (shown disabled with "Not available" on Android — ARP is not exposed to apps)

## Downloads
- `moustache-ping-v1.0.8-arm64.apk` — most phones (recommended)
- `moustache-ping-v1.0.8-armeabi-v7a.apk` — older 32-bit devices
